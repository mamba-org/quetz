`Issues & Ideas <https://github.com/TheSnakePit/quetz/issues>`__

:mod:`Quetz` is an open source server for conda packages built upon FastAPI with an API-first approach.

A Quetz server can have many users, channels and packages. With Quetz, fine-grained permissions on channel and
package-name level will be possible.

Quetz also comes with the quetz-client that can be used to upload packages to a quetz server instance.
